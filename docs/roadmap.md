
# Roadmap

Here is a list of planned features for this project:

- `@enabled_when_frozen` decorator
- Decorator that memoizes a instance method when the freezable instance is
  frozen
- Deep freezing

Here is a list of features being considered:

- Freezable collection types
- Threading safety

Want to suggest a feature? Post a [feature request][1] on the Github repo.

[1]: https://github.com/ederic-oytas/python-freezable/issues/new/choose
